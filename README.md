# 🍴 Pradeep Kitchen

A modern food ordering web application built with **Next.js**, **React**, **TypeScript**, and **Tailwind CSS**. The platform allows users to browse food items, explore recipes and blogs, view food details, add items to the cart, and place orders through a clean and responsive interface.

## 🚀 Features

* Modern responsive UI
* Browse food categories and food items
* Food detail pages
* Shopping cart functionality
* Blog section with dynamic routing
* Dark/Light theme support
* Search and filtering options
* Mobile-friendly design
* Static site deployment support
* Optimized image handling

## 🛠️ Tech Stack

### Frontend

* Next.js 15
* React 18
* TypeScript
* Tailwind CSS

### UI & Animations

* AOS (Animate On Scroll)
* React Slick
* Next Themes

### Utilities

* Axios
* Date-fns
* React Hot Toast

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/Pradeepkumar0018/Pradeep-Hotels.git
```

Move into the project directory:

```bash
cd Pradeep-Hotels
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```

## 🔨 Build Project

```bash
npm run build
```

Start production build:

```bash
npm start
```

## 🌐 Deployment

This project is configured for deployment on:

* Vercel
* Netlify
* GitHub Pages (Static Export)

Deploy using:

```bash
vercel
```

or connect the repository directly through the Vercel dashboard.
